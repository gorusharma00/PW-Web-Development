HTML Tags:

  1. Anchor tag: This tag creates the hyperlink to web pages, files, email addresses, telephone numbers, or anything else a URL can address.

  2. Heading Tag(h1- h6): The h1 to h6 tags are used to define the html headings. h1 defines the most important heading and h6 defines the least important heading.

  3. Paragraph Tag(p): This tag is used to defines the Paragraph.

  4. u: This tag represents that some text is unarticulated and styled diffrently from normal text.

  5. mark: This tag is used to highlight the some part of text.

  6. Ordered List tag(ol): This tag is used to represents the ordered list of items

  7. Unordered List tag(ul): This tag is used to represents the unordered list of items

  8. del: This tag represents deleted text from the document.

  9. strong: This tag indicates that its contents have strong importance, seriousness, or urgency.

  10. small: This tag defines the smaller text like copyright and other side comments.

  11. sup: This tag defines the superscript text.

  12. sub: This tag defines the subscript text.
