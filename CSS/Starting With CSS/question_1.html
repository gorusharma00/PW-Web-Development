There are three ways to add CSS:

1.Inline CSS

Inline CSS involves using the style attribute to apply specific styles directly to an HTML element. This means that the styles are defined withi the element's opening tag, allowing you to add a unique look and feel to that particular element without the need for a separate CSS file or style element in the document.

2.Internal CSS

Internal CSS, also known as embedded CSS or style tag CSS, allows you to define CSS rules directly within the HTML document itself. n. This approach provides a more localized and self-contained solution for applying styles to specific elements on the page.

3.External CSS

External CSS is a widely-used method of styling HTML documents. It involves creating a separate CSS file that contains all the styles for a website. This file is then linked to the HTML file.

If there are two or more CSS rules that point to the same element, the selector with the highest specificity value will "win", and its style declaration will be applied to that HTML element.

    Cascading Order:
    1. Inline CSS (Highest Specificity)
    2. Internal CSS
    3. External CSS (Lowest Specificity)
